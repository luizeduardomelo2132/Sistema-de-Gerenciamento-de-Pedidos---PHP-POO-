Sistema de Gerenciamento de Pedidos desenvolvido em PHP utilizando Programação Orientada a Objetos (POO).

O projeto simula uma empresa de vendas online, permitindo cadastro de produtos e clientes (comum e premium), criação de pedidos com múltiplos itens, cálculo automático do valor total e aplicação de descontos conforme o tipo de cliente.

O sistema aplica conceitos como encapsulamento, herança, polimorfismo, classe abstrata e modificadores de acesso, com as regras de negócio centralizadas nas classes.

Projeto acadêmico com foco na prática de POO em PHP.
